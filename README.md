# Approachlist

## プロジェクト概要

このプロジェクトは、営業活動で使用するアプローチリストを作成・管理するためのものです。
見込み顧客や既存顧客に対して、誰に・いつ・どのようにアプローチするかを整理し、営業活動の抜け漏れを防ぎ、継続的な成果につなげることを目的としています。

## 目的

- 営業対象を一覧化し、優先順位を明確にする
- アプローチ状況を可視化し、進捗を管理しやすくする
- 担当者ごとの活動履歴を蓄積し、次の行動につなげる
- 営業活動の標準化と効率化を進める

## 想定する活用内容

- 見込み顧客のリストアップ
- 顧客ごとの基本情報の整理
- 営業アプローチの履歴管理
- 次回アクションの設定
- 優先度や確度に応じたフォロー対象の整理

## 管理したい主な項目

- 企業名
- 担当者名
- 連絡先
- 業種
- 地域
- アプローチ手段
- 初回接触日
- 最終接触日
- 現在のステータス
- 次回アクション
- 優先度
- 備考

## 期待される効果

- 対応漏れや重複アプローチの防止
- 営業活動の見える化
- フォローアップの精度向上
- 案件化率の改善
- チーム内での情報共有の円滑化

## 今後の展開イメージ

今後はこのプロジェクトをベースに、営業アプローチリストのテンプレート作成、運用ルールの整理、更新フローの整備を進める想定です。
必要に応じて、スプレッドシートやWebアプリとしての管理方式にも拡張できます。

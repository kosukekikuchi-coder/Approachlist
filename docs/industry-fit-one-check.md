# industry_fit=1 実地確認メモ

## 目的
厳しめに定義した `industry_fit=1` を、実データ企業に対して小さく確認する。

## 今回確認した企業

### 1. 秋田印刷製本株式会社
- 判定: `industry_fit=1`
- 根拠:
  - [秋田印刷製本株式会社 Works](https://www.akitainsatu.co.jp/works/)
  - [秋田印刷製本株式会社 公式サイト](https://www.akitainsatu.co.jp/)

確認できたこと:
- 企画・デザイン・販売促進を明示している
- 「企業・店舗のイメージアップ」「販促ツールとして活用できるグッズ」と明示している
- 印刷だけでなく、販促支援文脈で顧客に提案する立場が見える
- まいぷれとの接続を公開情報で説明しやすい

判断:
- 地域の中小店舗・中小企業に対する販促支援との接続が明確であり、`industry_fit=1` の厳しめ基準を満たすと判断した

### 2. 株式会社トラパンツ
- 判定: `industry_fit=1` にはまだ上げない
- 根拠:
  - [株式会社トラパンツ 会社概要](https://www.torapants.co.jp/company)
  - [サポート・メンテナンス](https://www.torapants.co.jp/web/support)

確認できたこと:
- Web制作、運用管理、Web広告、DTP、販促イベント企画などは明確
- 顧客支援の立場も見える

見送り理由:
- 地域の中小店舗を主要顧客として持つことまでは、今回確認した公開情報だけでは断定しづらい
- したがって、現時点では `0.5` 相当の有望候補として扱うのが安全

### 3. 有限会社アイズブライト
- 判定: `industry_fit=1` にはまだ上げない
- 根拠:
  - [アイズブライト トップページ](https://aizbright.com/)
  - [アイズブライト バルーン](https://aizbright.com/balloon/)

確認できたこと:
- ブライダル、演出、バルーン装飾、開店祝い・周年祝い対応は確認できる

見送り理由:
- 地域の中小店舗を既に顧客として持ち、まいぷれを横展開しやすい立場とまではまだ言い切れない
- 現時点では `0.5` 以下に留めるのが安全

## 今回の結論
- `industry_fit=1` に上げたのは **秋田印刷製本株式会社** のみ
- 他の候補は、相性がありそうでも厳しめ基準にはまだ届かないため据え置いた

## 補足
- `industry_fit=1` は「本命候補」なので、今後も少数ずつ慎重に確認する
- `akita_company_details.csv` では `industry_fit=1` を反映済み
- ただし、現時点の `akita_company_master.csv` ではスコア反映の再確認が必要
